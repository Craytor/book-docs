In this chapter, we will learn the differences between client-side and server-side scripting, learn Javascript, and also take a brief look at jQuery, a Javascript library. Each type of scripting, client-side or server-side scripting, has its own use case. For example, if you are wanting to make the user's browsing experience interactive, you would want to use client-side scripting - if you want to do tasks such as sending emails, server side scripting would be needed. If you want to make something interactive, you should go with client-side scripting, but if you need to access results and display them on a webpage, you should use server-side scripting.
